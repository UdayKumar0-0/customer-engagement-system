# Customer Engagement & Notification System

A full-stack event-driven platform that tracks user behavior and triggers personalized engagement actions.

## 🚀 Features

* User authentication (signup/login with hashed passwords)
* Event tracking (signup, login)
* Event-driven notification (personalized welcome message)
* Analytics dashboard with filtering
* MongoDB relations using populate()

## 🧠 Architecture

Frontend (React) → Backend (Node.js/Express) → MongoDB

## 🛠 Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB (Mongoose)

## 📊 Key Concept

Implements an event-driven system where user actions generate events that can be used for analytics and engagement workflows.

## ▶️ Run Locally

1. Clone repo
2. cd backend → npm install → npm start
3. cd frontend → npm install → npm start
4. Add `.env` with MONGO_URI
